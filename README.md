# household-account-book

## using Tech stack

Ts, Js, React, Zustand, React-query, tailwindcss, react-router-dom

## 요구사항

[유저]
1. 한 유저는 한개의 가계부에 여러개의 입출금 내역을 가질 수 있다.
2. 입출금 내역을 삭제, 수정, 추가할 수 있다.
3. 한 유저는 여러개의 게시글을 가질 수 있다.
4. 한 유저는 여러개의 댓글을 가질 수 있다.
5. 한 유저는 다른 유저의 게시글에 댓글을 남길 수 있다.
6. 댓글은 삭제, 수정, 추가 할 수 잇다.

[가계부]
1. 유저는 각 1개의 가계부를 갖는다.
2. 가계부 작성시에는 날짜, 제목, 금액이 반드시 적혀야 한다.
3. 가계부는 입금과 출금내역으로 나뉜다.
4. 입금은 용돈, 보너스, 기타, 월급으로 나뉜다.
5. 출금은 식비, 기타, 유흥, 관리, 주거, 등등으로 나뉜다.
6. 한달, 3개월, 6개월, 1년단위로 총 소득 대비 지출금액을 확인할 수 있다.
7. 입금, 출금 카테고리별로 조회 가능하다.
8. 모든 조회는 날짜를 지정해서 원하는 날짜 사이의 금액을 확인가능하다.
